# Консольная библиотека (курсовой проект, Java ООП)

## Описание

Программа имитирует учёт **каталога** библиотеки (книги и журналы), **читателей** и операций **выдачи** и **возврата**. Управление через текстовое меню в консоли. Данные при старте заполняются демонстрационным каталогом; при желании вы можете расширить код и добавить сохранение в файл — для защиты достаточно текущей логики.

## Используемые технологии

- **Java** 17  
- Сборка: **Apache Maven** (опционально, если Maven установлен)  
- Кодировка UTF-8  

## Как запустить проект

### Вариант A: Maven

```bash
mvn compile exec:java
```

### Вариант B: только JDK (как в учебной среде без Maven)

Из корня репозитория (Windows PowerShell):

```powershell
$env:JAVA_HOME = "C:\Program Files\Eclipse Adoptium\jdk-17.0.18.8-hotspot"
javac -encoding UTF-8 -d target\classes src\main\java\com\course\library\*.java
java -cp target\classes com.course.library.Main
```

Подставьте свой путь к JDK при необходимости.

**Кодировка (Windows):** если русский текст в консоли отображается «кракозябрами», перед запуском выполните `chcp 65001` в том же окне терминала.

## Примеры использования

1. Запустите программу, откроется меню.  
2. **Пункт 1** — показать весь каталог (книги и журналы с количеством свободных экземпляров).  
3. **Пункт 2** — поиск, например по слову `Java` или по автору `Мартин`.  
4. **Пункт 3** — выдача: номер карты `1001`, ID позиции `B-001`.  
5. **Пункт 4** — возврат с теми же данными.  
6. **Пункт 6** — для `B-001` и `M-100` программа покажет **разный** максимальный срок выдачи (полиморфизм: книга 21 день, журнал 7 дней).  
7. **Пункт 7** — введите год, например `2019`, чтобы увидеть фильтр по году издания (разбор числа из строки и обработка ошибок ввода).

## Кратко для презентации (архитектура и ООП)

| Идея | Где в коде |
|------|------------|
| **Классы и объекты** | `Book`, `Magazine`, `Member`, `Library`, `Main` и др. |
| **Инкапсуляция** | Поля `private`, доступ через геттеры/сеттеры и методы домена (`LibraryItem`, `Member`, …). |
| **`this`** | В конструкторах и сеттерах (`LibraryItem`, `Book`, …). |
| **Наследование** | `Book` и `Magazine` расширяют абстрактный `LibraryItem`. |
| **Полиморфизм** | Переопределение `getMaxBorrowDays()`, `getCatalogLine()`, `matches()`; в `Library.listCatalogLines()` обход списка как `LibraryItem`. |
| **Интерфейс** | `Searchable` — контракт поиска; реализует `LibraryItem`. |
| **Абстракция** | Абстрактный класс `LibraryItem` с общим состоянием и абстрактными методами. |
| **Исключения** | Своё `LibraryException`, `try-catch-finally` в `Main`, `throw` в `Member` и при отсутствии экземпляров. |
